# CodeAlpha Secure Coding Review

## Objective

Perform a security audit on a Python Login System and identify vulnerabilities.

## Tools Used

- Python
- VS Code
- Bandit

## Vulnerabilities Found

- SQL Injection
- Plain Text Password Storage
- Input Validation Issues

## Fixes Implemented

- Parameterized Queries
- Password Hashing
- Input Validation

## Outcome

Successfully reviewed and secured the application by following secure coding best practices.
